Projects

TopThat is a mobile responsive web application that allows users to host videos and interact competitively within the TopThat social network. The app was built with Ruby on Rails, PostgreSQL, and AJAX, while integrating external technologies like Carrierwave, Amazon S3, Redis, and Sidekiq. Testing was conducted with RSpec, Shoulda Matchers, and Travis CI, and the app was deployed on Heroku


Overflow Your Boat is a light version of Stack Overflow with CRUD capabilities for creating questions, answers, and comments. The web application is built with Ruby on Rails, PostgreSQL, jQuery, and AJAX. It is tested using RSpec, Travis CI, and deployed on Heroku.
